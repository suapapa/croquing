Designed for Creative Focus

Built to address the frustrations of standard video-conferencing croquis sessions.

Bye, Compression. Hello, Detail.

Unlike video screen sharing which compresses images and introduces lag, Croquing loads original photographs directly onto every participant's device. Spot fine anatomy details and textures clearly.

Server-Authoritative Clock

Prevent premature sketches. The host sets each round length (1–60 minutes) before drawing begins, and an authoritative server timer keeps every artist in sync. When the round starts, the photo unlocks. When time runs out, it immediately vanishes.

Integrated Image Curation

Hosts can search and select curated figure images in seconds, directly inside the admin panel. Photos credit the original artists automatically, matching standard community guidelines.

Fair & Unpredictable Order

To preserve the challenge of quick gesture drawing, the session sequence is randomized. No artist knows what model or pose is coming up next until the countdown timer strikes zero.
